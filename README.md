
# Lógica do Jogo — Xadrez (versão básica)

Este projeto implementa uma versão simplificada da lógica do jogo de xadrez em Python.  
Objetivo: demonstrar movimentação de peças e estrutura de tabuleiro como exercício didático.

## Pecas implementadas  
- Rei  
- Torre  

## Funcionalidades  
- Representação do tabuleiro 8x8  
- Posicionamento inicial de peças  
- Movimentação válida de Rei e Torre (sem regras de xeque, xeque-mate, roque etc.)  
- Mover peças usando coordenadas (origem → destino)  
- Exibir o tabuleiro no terminal
